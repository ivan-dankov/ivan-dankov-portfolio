# Ivan Dankov Portfolio

This is the codebase for Ivan Dankov's personal portfolio website.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## stack

- React
- Vite
- Tailwind CSS
